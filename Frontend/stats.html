Score Formula

Score(t)  =  Σ (+50 per target collected)
          −  Σ (terrain_penalty per road segment traversed)
          −  100 per enemy contact (per agent)

Terrain penalties accumulate continuously while moving:
  Footway / Path → −5   |   Service road → −3
  Residential    → −2   |   Motorway    → −8
  Unknown/Other  → −1 to −4 depending on highway type

In multi-agent mode:
  Score = sum of all agent scores.
  Each agent independently accumulates targets, terrain costs, escape events.
  When a target is collected it is removed for ALL agents (cooperative dedup).
  Agents are pre-assigned distinct targets (greedy by proximity) to maximise
  spatial coverage and minimise redundant routing.

Escape vs Brave decision:
  Aggressive enemy within 1.2× detection_radius?
    → BRAVE if robot arrives at nearest target before enemy  (score stays ↑)
    → ESCAPE otherwise (score plateaus or dips until threat passes)

Score gradient interpretation:
  Steep upward steps  = target collected
  Gradual slope down  = terrain/energy cost
  Sharp drop          = enemy contact (−100)
  Flat section        = escape detour, no targets in range
